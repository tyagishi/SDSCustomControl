# SDSCustomControl

Convenient control collection

## LongPressableButton
Button that can handle not only tapAction but also long-press

### example
```
LongPressableButton(tapAction: {
    // action for tapped
}, longPressDetected: {
    // action for long press "detected" 
}, longPressAction: {
   // action for long pressed
}, label: {
   // label for button
})

```
